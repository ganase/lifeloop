lifeloopについて

lifeloopは、登録した場所への到着や離脱をきっかけに、その場で取り組みやすい行動を端末内のローカル通知として表示するiOSアプリです。

アカウント登録、広告、外部分析SDK、独自サーバーへのデータ送信はありません。場所、行動、通知履歴などのデータは利用者の端末内に保存されます。
